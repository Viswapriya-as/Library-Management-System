# Library Management System

A console-based Library Management System built in core Java.

## Features
- Add and manage books and members
- Issue and return books with real-time availability tracking
- Custom exception handling for invalid operations (e.g. issuing an already-issued book)
- Data persistence using file I/O (library_data.txt)

## Tech Stack
Java, OOP, Collections (ArrayList, HashMap), Exception Handling, File I/O

## How to Run
1. Clone the repo
2. Compile: `javac *.java`
3. Run: `java LibraryManagementSystem`
